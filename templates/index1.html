Summer Internship on Machine Learning


AICTE Sponsored Neural Network & Deep Learning Lab at Poornima Institute of Engineering & Technology has organised an “Summer Internship-2022” on Machine Learning & Data Science at PIET Jaipur, India from 27 June -8 August, 2022 (45 Days) in association with Upflairs Pvt limited. The Internship is for regular UG/ PG level students of Engineering. The internship program is designed for all students of BTech to understand and implement AI, Data Analytics, Data Science, Machine Learning (ML), Natural Language Processing techniques in different fields. It was conducted by Mr. Deepak Moud, HoD, CSE, PIET and Up flairs Trainers, GPU support was provided to develop high end projects.





Machine Learning models on Income Classification


The Project is based on designing a website which has different models that are capable of classifying that whether the income of an individual is less than 50K or more than 50K. The classification is based on the inputs given by the user. The models deployed which can be used for classification are mentioned below.



Decision Tree

Decision Tree is a Supervised learning technique that can be used for both classification and Regression problems, but mostly it is preferred for solving Classification problems.

Support Vector Machine

Support Vector Machine or SVM is one of the most popular Supervised Learning algorithms, which is used for Classification as well as Regression problems.

Random Forest

Random Forest is a popular machine learning algorithm that belongs to the supervised learning technique. It can be used for both Classification and Regression problems in ML.

Linear Regression

Linear Regression is a supervised machine learning algorithm where the predicted output is continuous and has a constant slope. It's used to predict values within a continuous range.

K-Nearest Neighbor(KNN)

KNN works by finding the distances between a query and all the examples in the data, selecting the specified number examples (K) closest to the query, then votes for the most frequent label

Naive Bayes Classifier

Naive Bayes is a classification algorithm for binary (two-class) and multi-class classification problems. The technique is easiest to understand when described using binary or categorical input values.
